Tools
In this section we will briefly cover the tools and platforms which we will use during the next sessions. Some of the facilities might be necessities and others just an extra thing which will make your life easier.
Code Editor
Whatever code you will be writing you need an editor to write it in. For sure you are familiar with Google docs, which is used for text editing, so consider code editors as the same except that they are used for code altering. The editors that we suggest are:

Visual Studio Code

Sublime Text 3

Atom

You can find the instructions for installations, and corresponding guides. Just pick one of them which you like the most and create your files there.
Every file that you will be creating should have a name and an extension, for example “index.html” where the name is “index” and “html” is the extension(format). So for each type of file you should indicate the extensions:

file_name.html - for HTML file

file_name.css - for CSS file

file_name.js - for JS file

file_name.py - for Python file

file_name.ipynb - for Jupyter/Colab (notebook) file

After creating an HTML file you can execute it by opening the corresponding file in a web browser
Note: CSS and JS files can’t be executed separately without being included in an HTML file.
Python
For the instructions on how install Python on your machine , after you have done it, you should be able to access the Python alias in your terminal/cmd. To execute a Python file you can use terminal by typing in: 
python3 file_name.py
Another way of executing Python files is by using “Notebooks”such as Jupyter or Colab notebooks, they both run on “ipynb” extension files. The advantage of these files is that you can run codes block by block. And the results of execution stay there, while in the “py” file you run the entire program. You need to install Jupyter because it runs locally, this being said it uses your computer’s resources, while Colab is running on “cloud”. It’s built by Google and they provide pretty powerful servers for free. So we definitely recommend Colab over Jupyter.
